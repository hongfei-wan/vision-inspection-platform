# WPF 视觉检测上位机平台 - 项目架构设计

## 系统架构图

```
┌─────────────────────────────────────────────────────────────┐
│                  表现层 (Presentation Layer)                 │
│                    WPF UI + MVVM Pattern                    │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│                  业务逻辑层 (Business Layer)                 │
│  CameraModule | AlgorithmManager | CommunicationModule     │
│  DataModule | WorkflowEngine | ReportModule                │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│                   算法层 (Algorithm Layer)                   │
│  HalconEngine | OpenCVEngine | YOLOEngine | HalconDLEngine │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│                基础设施层 (Infrastructure Layer)             │
│  Database | FileSystem | Logger | Configuration | Utils    │
└─────────────────────────────────────────────────────────────┘
```

## 核心特性

- **分层架构**: 清晰的职责分离
- **依赖注入**: 使用 DI 容器管理生命周期
- **接口驱动**: 基于接口的多实现支持
- **异步处理**: 保证 UI 响应性
- **可扩展性**: 插件架构支持扩展

## 模块说明

### 相机模块
- 支持多品牌相机
- 实时预览采集
- 参数动态配置
- 多相机管理

### 算法模块
- 统一的算法接口
- 参数动态配置
- 性能监控
- 结果缓存

### 通讯模块
- Modbus 协议
- TCP/IP 协议
- 串口通讯
- OPC-UA (可选)

### 数据模块
- 关系型数据库
- 图像文件存储
- 数据导出 (Excel/CSV/PDF)
- 高级查询功能

### 工作流引擎
- 流程设计
- 条件判断
- 循环处理
- 错误处理
